# VKBotForGroupChat

Заготовка для VK-бота.
Бот для общения в чате сообщества

Схема проекта:
1) Пользователь пишет вк-боту
2) Если сообщение пользователя содержит запрещенный слова, пользователь получает предупреждение
3) Если с сообщением все ок, пользователь получает ответ
+ в чате доступны кнопки (с переходом на нужные ссылки)

Для запуска:
1) Зайти в сообщество, перейти во вкладку управление > работа с API > получить ключ
2) Сохранить ключ в виде строки в переменную "VK_BOT_TOKEN" в предварительно созданном файле config.py 
3) В той же вкладке "работа с API" проставить нужные галочки во вкладке "Long Poll API"
3) Запустить приложение (app.py)
4) Бот активирован!
